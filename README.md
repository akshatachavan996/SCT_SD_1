# SCT_SD_1
A modern and responsive Temperature Converter built using HTML, CSS, and JavaScript. Converts temperatures between Celsius, Fahrenheit, and Kelvin with a clean UI, input validation, and instant conversion. Developed as Task 1 for the SkillCraft Technology Software Development Internship.

---

## 🚀 Live Features

✨ Convert between:
- 🌡️ Celsius (°C)
- 🔥 Fahrenheit (°F)
- ❄️ Kelvin (K)

---

## 🎯 Key Features

- 🔄 Swap between units instantly
- ⚠️ Inline error handling (no alerts)
- 📋 Copy result to clipboard

- 📱 Fully responsive design

## 📚 Conversion Formulas

### Celsius → Fahrenheit
```
°F = (°C × 9/5) + 32
```

### Fahrenheit → Celsius
```
°C = (°F − 32) × 5/9
```

### Celsius → Kelvin
```
K = °C + 273.15
```

### Kelvin → Celsius
```
°C = K − 273.15
```
